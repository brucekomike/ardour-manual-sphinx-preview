# Adding Local Clip Libraries
You can add as many sample library locations as you like and easily
switch between them using the drop-down list at the top of the Clips
browser.

There are two ways to add new library location. The first one is by
using the [Edit Sample Library Path]{.kbd .window} window directly:

1.  In the [Clips]{.dfn} tab of the sidebar, click the top drop-down
    list and choose [Edit]{.kbd .menu}.
    <figure class="right">
    <img src="/images/clips-locations-in-the-dropdown-list.png" width="400"
    alt="Clips&#39; locations in the drop-down list" />
    <figcaption aria-hidden="true">Clips' locations in the drop-down
    list</figcaption>
    </figure>
2.  In the newly opened dialog called [Edit Sample Library Path]{.kbd
    .window} click [Add]{.kbd .button} and locate the folder that
    contains samples. If there are subfolders, choose the parent folder
    that contains all of them.
3.  Repeat for all other sample libraries.
4.  Click [OK]{.kbd .button} to apply changes.

Alternatively, you can drop a folder from a file manager into the list
of clips in the Clips browser. This will add a new clip library location
stored across sessions.

Now when you click the drop-down list at the top of [Clips]{.dfn}, you
will see all the paths you added.

Should you need to delete any path you previously added, open the [Edit
Sample Library Path]{.kbd .window} dialog again, select the path and
click [Delete]{.kbd .button}.
