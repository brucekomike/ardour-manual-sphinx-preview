# Adding Pre-existing Material
There are several ways to importing an audio or MIDI file into a
session:

- Main menu: [Session \> Import]{.kbd .menu}
- Region List context menu: [Import To Region List]{.kbd .menu}
- Track context menu: [Import Existing Media]{.kbd .menu}

These methods are all equivalent: they open the [Add Existing
Media](@@import-dialog) dialog.

Finally, files can also easily be imported into a project by dragging
and dropping a file from some other application (e.g. the system\'s file
manager). Files can either be dragged onto the [Region List]{.dfn}, into
the desired [track]{.dfn} or into an empty space in the editor track
display.

The file will be imported and copied into the session, and placed at the
position where the drag ended.
