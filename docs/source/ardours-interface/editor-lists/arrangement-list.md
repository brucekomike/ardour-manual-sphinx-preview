# The Arrangement List
The Arrangement list shows all range sections in the session, with start
and end time for each of them in the time domain of the first clock.

![Arrangement list](/images/arrangement-list.png){style="width:50%;"}

Selecting a range section in the list opens a small window below that
display range section timing: end, start, and duration. Its time domain
can be selected in the right-click menu.

![Selected range section in the Arrangement
list](/images/arrangement-list-selected-section.png){style="width:50%;"}

## Copying and pasting range sections

An existing range section can be easily copied and pasted using this
list editor. Simply pick a range section and drag it up or down to
insert between two other existing sections, before the first section, or
after the last section.

## Renaming range sections

To rename an existing range section, either double-click its name or
right-click it and select the [Rename the selected Section]{.kbd .menu}
menu item, then input the new caption of the range section, and press
[Enter]{.kbd} to confirm.

To cancel the renaming before the confirmation, press the [Esc]{.kbd}
key.

## Removing range sections

To remove an existing range section, either press the [Del]{.kbd} key or
select the [Remove the selected Section]{.kbd .menu} menu item.
