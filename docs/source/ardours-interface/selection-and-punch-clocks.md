# Selection Clocks
## The Selection Clocks {#selection_clocks}

![The range clocks](/images/selection-clocks.png)

The current [selection range]{.dfn}, as set with the [Range Mode
tool](@@toolbox), is displayed in these three clocks: start of the
range, end of the range, and length.

Clicking on the range clocks will locate to either the beginning or end
of the punch range.

[Right]{.kbd .mouse} clicking on any of the clocks brings up a context
menu allowing to change the type of time display between the [4 clock
modes](@@editing-clocks), and to copy the selected clock\'s time to the
clipboard.
