# Playhead Options
![The playhead options](/images/playhead-options.png)

Those two buttons control the behaviour of the playhead:

- [Follow Playhead]{.dfn} enables the mode where Ardour automatically
  scrolls the timeline to follow the playhead and always display it on
  the screen.
- [Follow Range]{.dfn} enables the mode where making a range selection
  moves the playhead to the start of the range.
