# Common Region Edit Operations
This section covers a set of [region editing operations]{.dfn} that are
likely to be used often while working on a session. Depending on work
habits (and experience of other [DAW]{.abbr
title="Digital Audio Workstation"}s), some of these operations will be
critical while others are used only rarely.

All of these operations can be carried out from the keyboard (see
[Default Keyboard Shortcuts](@@default-shortcuts) for a list).
Equivalent operations can be performed with the mouse in most cases.

Some of these operations make use of [the edit
point/range](@@edit-point-control) and [affect specific
regions](@@which-regions-are-affected).

  ------------------------------- -------------------------------------------------------------------------------------
  [Spot (Align)]{.kbd .menu}      Move selected regions to the edit point.
  [Split]{.kbd .menu}             Split selected regions at the edit point.
  [Trim Start]{.kbd .menu}        Adjust the start of selected regions to the edit point (or as close as possible).
  [Trim End]{.kbd .menu}          Adjust the end of selected regions to the edit point (or as close as possible).
  [Duplicate]{.kbd .menu}         Make a copy of each selected region and position it immediately after the original.
  [Crop]{.kbd .menu}              Truncate selected regions to the edit range.
  [Separate]{.kbd .menu}          Split selected regions at both ends of the edit range.
  [Set Fade In]{.kbd .menu}       Adjust selected audio regions\' fade in to end at the edit point.
  [Set Fade Out]{.kbd .menu}      Adjust selected audio regions\' fade out to end at the edit point.
  [Toggle Fade In]{.kbd .menu}    Turn selected audio regions\' fade in on or off.
  [Toggle Fade Out]{.kbd .menu}   Turn selected audio regions\' fade out on or off.
  [Play Region]{.kbd .menu}       Play session from the start of the earliest selected region.
  [Zoom To Region]{.kbd .menu}    Zoom horizontally so that the selected regions span the editor track view.
  [Set Sync Point]{.kbd .menu}    Set the sync point of all selected regions to the edit point.
  [Insert]{.kbd .menu}            Inserts the currently selected regions in the Region List at the edit point.
  ------------------------------- -------------------------------------------------------------------------------------
