# Microsoft Windows
[Microsoft Windows]{.dfn} is officially supported since version 5.0
(2016).

## Installing Ardour

1.  Download the latest windows build from [the download
    page](https://community.ardour.org/download).
2.  Run the installer and follow the prompts.

## How to help

- Hang out in [#ardour](https://web.libera.chat//#ardour) on Libera. One
  may ask questions there and if possible, answer questions that others
  have.
- Keep an eye on the [Windows
  forum](https://discourse.ardour.org/c/installation-configuration/windows)
  and contribute to the discussions there.
- Update this manual via pull requests on
  [GitHub](https://github.com/Ardour/manual).
