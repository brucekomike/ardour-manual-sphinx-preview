# Creating MIDI Regions
Although [recording MIDI](@@midi-recording) on an existing track is a
common way to create new MIDI regions, it is often desirable to create
new MIDI regions as part of the editing and/or arranging process.

A new MIDI region can be created with the mouse by entering [Draw
Mode](@@toolbox) (entered by pressing the [d]{.kbd} key or clicking on
the Draw tool) and then [left]{.kbd .mouse}-clicking anywhere in an
existing MIDI track; this will create a region that is one bar long. If
longer regions are desired, they can be created by [left]{.kbd
.mouse}-clicking anywhere in an existing MIDI track and dragging the
mouse until the region is the desired length.

Once created, all the typical methods of editing audio regions will work
the same as they do for MIDI regions. See [Editing Regions and
Selections](@@editing-regions-and-selections) for more details.
