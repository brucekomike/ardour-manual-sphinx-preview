# Bus Controls
![A bus header
section](/images/typical-bus-controls.png){alt="bus controls"}

The bus\' header is very similar to the [audio track
header](@@audio-track-controls), minus :

- the playlist button, as a bus doesn\'t have any playlists or regions
  by itself, it is only a pipe to route audio or midi through
- the record button, for the same reason.

For more information about the bus concept, see [Understanding basic
concepts](@@understanding-basic-concepts-and-terminology#busses).
